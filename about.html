Sobre este directorio

Un directorio abierto y mantenido por la comunidad para descubrir proyectos, herramientas y recursos del ecosistema Bitcoin y blockchain.

¿Qué es esto?

El Directorio BTC es una colección curada de proyectos, herramientas y recursos relacionados con Bitcoin y blockchain. El objetivo es que cualquier persona pueda encontrar fácilmente lo que busca: desde wallets y nodos hasta herramientas de análisis on-chain, recursos educativos o proyectos de privacidad.

No hay algoritmos de recomendación, ni publicidad, ni tracking. Solo una lista honesta mantenida por la comunidad.

¿Cómo funciona?

  • Cualquiera puede enviar un proyecto desde el formulario de la web.
  • El formulario crea un Issue en GitHub y se revisa manualmente para mantener la calidad.
  • Cada envío genera automáticamente una Pull Request para revisión, sin publicar nada directamente en la rama principal.
  • Una vez aprobada esa PR, el proyecto queda añadido al archivo data/projects.json en el repositorio.
  • Los datos son totalmente abiertos y accesibles en GitHub — transparencia total.
  • La web sigue siendo estática y solo usa una función mínima para recibir envíos de forma segura antes de generar la PR.

¿Cómo colaborar?

  • Envía un proyecto que conozcas usando el formulario.
  • Reporta enlaces rotos o datos incorrectos abriendo un Issue en GitHub.
  • Si sabes programar, puedes contribuir directamente vía Pull Request.
  • Apoya con sats el mantenimiento desde la página de donativo.
