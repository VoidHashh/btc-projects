Abrir wallet

El QR y el enlace usan la URI bitcoin:?lno=... para abrir la donacion desde wallets compatibles con BOLT12.

  • Escanea con una wallet Lightning compatible con BOLT12.
  • Si tu wallet no abre el enlace, copia manualmente el codigo BOLT12.
  • Si tu app todavia no soporta BOLT12, usa el metodo on-chain como respaldo.

Escanealo desde una wallet Lightning moderna. Si no abre bien, usa el boton de copiar.

Compatibilidad

Esta implementacion esta pensada para wallets con soporte BOLT12. La compatibilidad aun depende de que la wallet emisora entienda el formato o la URI bitcoin:?lno=....
